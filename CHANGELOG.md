# Changelog

All notable changes to this package will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-04-28

### Added
- Initial release.
- Style library asset with name, background color, side icon, bold, and text color.
- Per-scene database persisted next to each scene as a `ScriptableObject` asset.
- Right-click context menu (`GameObject → Add Style...`) with a style picker popup.
- Manager window at `Tools → Hierarchy` with Styles and Scene Entries tabs.
- Hover and selection state tinting for styled rows.
- Background that covers the foldout arrow on parent entries.
- Automatic and manual pruning of stale entries.
